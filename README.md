#  Finance Analytics Dashboard | Power BI

##  Project Overview

This project presents an interactive Finance Analytics Dashboard developed in **Power BI** to analyze financial transactions, customer behavior, transaction fees, taxes, and key business performance indicators.

The dashboard provides an end-to-end analytical view of financial data, enabling users to monitor KPIs, identify trends, evaluate customer segments, and explore transaction-level insights.

---

##  Business Objective

The objective of this dashboard is to help stakeholders answer business questions such as:

- What is the total financial transaction value?
- How many transactions were processed?
- Which customer segments generate the highest fees?
- How do transaction fees change over time?
- Which states contribute the highest transaction fees?
- What is the distribution of transaction status?
- How do transaction patterns differ by gender?
